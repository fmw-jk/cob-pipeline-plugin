Durch Auswahl dieser Option erhalten Sie die Möglichkeit das Repository zusätzlich zu der "Priorisierten Umgebung" in weiteren Systemumgebungen zu bauen und zu testen. Die unten erscheinende Matrix stellt alle unterstützten Kombinationen aus Ubuntu Release Versionen und Architektur dar. Wählen Sie dort die gewünschten Umgebungen aus. Beachten Sie, dass für diese nicht alle Pipeline Jobs ausgeführt werden sondern nur der Build Job. Wenn Sie für eine weiter Kombination aus Ubuntu Version und Architekture die gesammte Pipeline durchlaufen wollen, erstellen Sie bitte eine weitere Instanze des Repositorys (mit Suffix).
