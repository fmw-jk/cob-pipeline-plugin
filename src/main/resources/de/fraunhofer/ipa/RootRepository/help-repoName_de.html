Name des GitHub-Repositorys, welches getestet werden soll. Die Existenz wird validiert.
When Sie die Eingabe beginnen werden nur die Repositories des Standard Forks/Besitzers angezeigt.

Falls Sie ein Repository eines anderen Forks/Besitzers konfigurieren und die dafür verfügbaren Repositories sehen möchten:
• gehen Sie zu 'Weitere Repository-Optionen'
• geben Sie dort den Forkbesitzer ein
• kehren Sie zurück und sehen Sie bei der Eingabe eine Liste der Repositories
